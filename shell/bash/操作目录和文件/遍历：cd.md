# 遍历：cd
> change current working directory
>
>> 改变当前工作目录.
>>
>>> 1. 特别基础和底层，man不到.
>>> 2. 无选项.

<br><br>

- **特殊路径符号**：

> 在/处继续cd ..结果仍位于/

| 特殊路径 | 符号 |
| --- | --- |
| ~ | **当前登录用户**的home目录，如**/home/Peter/** |
| . | 当前目录 |
| .. | 当前目录的**上级目录** |

- **命令用法**：

| 用法 | 语法 | 说明 |
| --- | --- | --- |
| 特殊用法：无参 | cd | 切到~ |
| 正常用法：参数可以是**绝对路径**或者**相对路径** | cd **路径** | 切到指定路径 |
