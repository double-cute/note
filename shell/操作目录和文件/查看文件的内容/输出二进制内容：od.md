# 输出二进制内容：od
> 之前介绍的cat、tac、nl、head、tail、more、less只能用来查看文本文件.
>
> - 如果用它们查看二进制文件（可执行文件、数据文件等）会输出乱码.
>
>> od命令可以输出任何文件的二进制内容，因此不管文本文件还是二进制文件都能查看.
>>
>>> 1. 是Octal Dump的缩写.
>>> 2. 因此从字面上可以直观的看出它 **输出内容默认以八进制表示**.
>>> 3. 但可以随意调整输出的 **内容和地址** 的格式（十进制、八进制、十六进制）.
>>>    - 其中，内容还可以用浮点数表示！
