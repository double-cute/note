# 安装
> Redis是POSIX系统的源生软件，因此可以轻松地在Linux/Unix、Mac OS X中编译安装.
>
>> - Redis并不支持Windows，因为：
>>    1. Redis主要应用在服务器领域，相比之下Windows不那么重要.
>>    2. Redis本身使用了大量的系统调用（写时复制等），移植到Windows成本太大.
>>
>> - 但Windows向Redis提交了一个兼容性补丁，并自己开发了一个Redis分支.
>>    - 但 **只能通过Cygwin的方式编译安装**.
