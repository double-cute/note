# 鸡尾酒排序
> 即 **双向冒泡排序**.
>
>> 只不过就是往后往前来来回回，最大沉底，最小浮顶.
>>
>>> 就是单向冒泡的花式玩弄，没有任何区别，复杂度和单向冒泡完全一样.

<br><br>

### 实现：

<br>

```C++
void
cocktail_sort(int arr[], int len) {
    bool exchHappened;
    int top = 0, bottom = 0;  // number already at top & bottom
    int left = len;  // number left to sort
    int i;
    while (left > 1) {
        exchHappened = false;
        i = top;
        while (i < len-1-bottom) {
            if (arr[i] > arr[i+1]) {
                swap(arr, i, i+1);
                exchHappened = true;
            }
            ++i;
        }
        ++bottom;
        if (!exchHappened || --left <= 1)
            return;

        exchHappened = false;
        i = len - 1 - bottom;
        while (i > top) {
            if (arr[i] < arr[i-1]) {
                swap(arr, i, i-1);
                exchHappened = true;
            }
            --i;
        }
        ++top;
        if (!exchHappened || --left <= 1)
            return;
    }
}
```
